# Code of Conduct

The aitap project adopts the **[Contributor Covenant, version 2.1][cc]** as
its Code of Conduct. The full text — covering expected behavior,
enforcement responsibilities, scope, and the four-tier enforcement
ladder — is maintained at the canonical URL below and applies in full to
every space this project operates in (issue tracker, pull requests,
discussions, chat, and any project-affiliated event).

[cc]: https://www.contributor-covenant.org/version/2/1/code_of_conduct/

## Scope

This Code of Conduct applies within all community spaces and also when
an individual is officially representing the community in public spaces.
"Community spaces" includes:

- The GitHub repository (issues, pull requests, discussions, code
  review).
- Any community chat or forum officially linked from the project README.
- Any in-person or virtual event organized under the project's name.

## Reporting

If you experience or witness behavior that violates this Code of
Conduct, please report it privately to the maintainers by emailing
**conduct@aitap.dev**. Reports will be reviewed and investigated
promptly and fairly. All maintainers are obligated to respect the
privacy and safety of the reporter.

If a maintainer is the subject of a report, please send the report to
another maintainer of your choice; do not send it to the subject.

## Enforcement

Maintainers will follow the **Community Impact Guidelines** described in
[Contributor Covenant 2.1][cc] when determining the consequences of any
violation. The four enforcement levels — Correction, Warning, Temporary
Ban, Permanent Ban — and their applicability are defined in that
document.
